## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to the repository owner or open a private security advisory.

Please do not open a public issue for security vulnerabilities. We will triage and address all security reports promptly.

### Responsible Disclosure
- Provide detailed steps to reproduce the vulnerability.
- Allow us a reasonable amount of time to fix the issue before making it public.
- Avoid using the vulnerability in any way that might harm the project or its users.
